# Imágenes de Marcas de Calderas

Coloca las imágenes de los logos de las marcas en esta carpeta.

## Imágenes requeridas

- `baxilogo.webp` - Logo de Baxi
- `peisalogo.webp` - Logo de Peisa
- `aristonlogo.webp` - Logo de Ariston
- `eutermalogo.webp` - Logo de Euterma
- `caldaialogo.webp` - Logo de Caldaia

## Especificaciones

- **Formato**: WebP (preferido) o PNG
- **Tamaño**: 136x96px (o proporción similar)
- **Fondo**: Transparente o blanco según el diseño del logo

## Uso

Una vez que agregues las imágenes, actualiza el archivo `src/data/brands.js` para usar imports en lugar de rutas públicas.
