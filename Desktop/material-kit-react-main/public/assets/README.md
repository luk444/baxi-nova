# Imágenes de Marcas de Calderas

Esta carpeta contiene las imágenes de los logos de las marcas de calderas que se muestran en el carrusel.

## Imágenes requeridas

Coloca las siguientes imágenes en esta carpeta (`public/assets/`):

- `baxi-logo.webp` - Logo de Baxi
- `peisalogo-C4Bl6OLG.webp` - Logo de Peisa (ya existe según el ejemplo)
- `ariston-logo.webp` - Logo de Ariston
- `eutermalogo-Ct0PIleP.webp` - Logo de Euterma (ya existe según el ejemplo)
- `caldaia-logo.webp` - Logo de Caldaia
- `roca-logo.webp` - Logo de Roca
- `junkers-logo.webp` - Logo de Junkers
- `vaillant-logo.webp` - Logo de Vaillant

## Especificaciones recomendadas

- **Formato**: WebP (preferido) o PNG
- **Tamaño**: 136x96px (o proporción similar)
- **Fondo**: Transparente o blanco según el diseño del logo
- **Calidad**: Alta resolución para pantallas Retina

## Nota

Las rutas de las imágenes están configuradas en `src/data/brands.js`. Si cambias el nombre de algún archivo, actualiza también el archivo de datos.
